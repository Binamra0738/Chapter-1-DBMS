SQL Commands
(Day 5)

SQL Commands are fundamental for database management:


CREATE

The CREATE statement is used to create a new table or database object.


DROP

The DROP statement is used to delete an existing table, index, or database object.


ALTER

The ALTER statement is used to modify an existing database object, such as a table.


ADD

The ADD statement is used to add new columns to an existing table.


UPDATE

The UPDATE statement is used to modify existing records in a table.


SET

The SET statement is used in conjunction with UPDATE to specify which columns and values to modify.


INSERT

The INSERT statement is used to insert new records into a table.


SELECT

The SELECT statement is used to retrieve data from a database.
