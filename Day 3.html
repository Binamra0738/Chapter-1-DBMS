DBMS Terminologies

Data

Data refers to raw facts and figures that have no meaning on their own.


Metadata

Metadata provides information about other data. It describes the characteristics, usage, and context of data.


Data Dictionary

A data dictionary is a centralized repository of metadata about data. It stores definitions and information about data elements and their structure.


Information

Information is processed data that has meaning and context. It is useful for decision-making and understanding.


Database

A database is a structured collection of data organized for efficient retrieval, storage, and management.


Table

A table is a structured representation of data in rows and columns within a database.


Tuple

A tuple (or row) is a single record or row in a database table that contains a specific set of related data fields.


Domain

A domain specifies the set of possible values for a given attribute or column within a table.


Key

A key is a field or set of fields in a database record that uniquely identifies the record in a table.
